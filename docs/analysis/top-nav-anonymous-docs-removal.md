## Problem

Im anonymen Modus ist `Docs` zwar weniger irreführend als das Zahnrad,
gehört aber funktional nicht zur Kernaufgabe der öffentlichen Explore-Ansicht.

Die gewünschte Zielstruktur ist deshalb:

- anonymer Modus: nur minimale öffentliche Hauptnavigation
- eingeloggter Creator-Modus: erweiterte Navigation bleibt verfügbar

## Varianten

### Variante 1: `Docs` im anonymen Modus ausblenden

Vorteile:
- Klarere öffentliche Explore-Navigation
- Weniger Ablenkung
- Noch weniger interaktive Elemente im anonymen Header

Nachteile:
- Öffentliche Dokumentation ist nicht mehr direkt über den Header erreichbar

### Variante 2: `Docs` im anonymen Modus belassen

Vorteile:
- Öffentliche Hilfe bleibt direkt erreichbar

Nachteile:
- Header bleibt breiter als nötig
- Explore wirkt weniger fokussiert

### Variante 3: `Docs` in ein anonymes Mehr-Menü verschieben

Vorteile:
- Dokumentation bleibt erreichbar

Nachteile:
- Zusätzliche Komplexität
- Erneut unnötiges Dropdown im anonymen Modus

## Entscheidung

Variante 1 ist die gewünschte und einfachste Lösung.
Im anonymen Modus bleibt nur die minimale Hauptnavigation sichtbar.
Die Creator-Navigation bleibt unverändert.
